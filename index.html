WDW Rides

This mobile app, nicknamed "WDW Rides", was the first mobile application I ever created. This was a project I created for my Mobile Development class and we were allowed to choose any topic / type of app to create. Creating an application about rides at Walt Disney World was the first thing to came to mind for me.

This app was created using Swift and it pulls in data from GitHub. The rides data is stored in a json file on GitHub as well as all of the images used for the application. It pulls in that information using an API call, translates the data, and uses the data for the application. This experience taught me a lot about Mobile Development, UI/UX Design, and API Calls. It also loads in websites within the app itself if the user chooses to view the website.

As an added bonus, whenever the phone shakes it displays Steamboat Willie!
